># Le markdown 
>C'est un langage code qui sert a mettre des texte en gras ou italique en mettant des asterisques. On peut aussi choisir la taille du texte en mettant des dièses. 

```
*italique*
**gras**
***grasItalique***
#Titre
##Sous-titre 
###Petit titre
####Texte
#####Petit texte
######Très petit texte
```
